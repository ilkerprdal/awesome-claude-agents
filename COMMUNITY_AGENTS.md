# Community Claude Code Agent Repositories 🌍

This directory lists community-contributed repositories containing **Claude Code agents** and **sub-agents**. These repositories range from massive collections of production-ready agents to specialized toolsets for specific workflows.

## 🏆 Top Collections (Must-Have)

| Repository | Agents | Description |
|---|---|---|
| **[VoltAgent/awesome-claude-code-subagents](https://github.com/VoltAgent/awesome-claude-code-subagents)** | 100+ | **The Gold Standard.** Massive collection covering full-stack dev, DevOps, Data Science, and Business. Formerly `0xfurai`. |
| **[davepoon/claude-code-subagents-collection](https://github.com/davepoon/claude-code-subagents-collection)** | 43+ | Includes a **CLI tool** (`bwc-cli`) and Web UI. Features specialized agents for Architecture, QA, Marketing, and more. |

| **[vizra-ai/claude-code-agents](https://github.com/vizra-ai/claude-code-agents)** | 59 | A massive squad of specialized agents covering Marketing, Design, DevOps, Business Strategy, and Research. |
| **[wshobson/agents](https://github.com/wshobson/agents)** | 48+ | Production-ready specialists with orchestration patterns and advanced workflows. |

## 💡 How to Use These Agents

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/username/repo-name.git
    ```
2.  **Copy the agents**:
    Move the `.md` files from the cloned repo into your local Claude Code agents directory:
    ```bash
    mkdir -p ~/.claude/agents
    cp repo-name/*.md ~/.claude/agents/
    ```
3.  **Restart Claude Code**:
    Run `claude` in your terminal.
4.  **Invoke**:
    Ask Claude to use the agent: "Use the [agent-name] to help me with..."
